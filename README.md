A execução desse projeto se estende por conexão por LAN e localhost

Para executar o jogo em localhost é necessário criar três terminais:
- No primeiro terminal abra o servidor com: python server.py;
- No segundo e terceiro terminais abra o cliente com: python client.py.

Após apenas clique no botão para conectar ao servidor com os dois clients.

Para executar o jogo por LAN é necessário ter pelo menos duas máquinas:
- A primeira máquina vai abrir o servidor e o client;
- A segunda máquina vai abrir apenas o client;
- No momento da conexão, insira o ip da rede onde os computadores estão conectados.
